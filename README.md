# Exercícios e Projetos em Java (Eclipse)

Este repositório reúne diversos exercícios e práticas de programação em **Java**, desenvolvidos no **Eclipse IDE**.  
O objetivo é consolidar conceitos de lógica de programação, estruturas de dados e orientação a objetos.

## Estrutura
Cada subpasta representa um projeto independente, por exemplo:
- `ExercicioVetores`
- `ExercicioArrayList`
- `PraticaCalculadoraSwitchCase`
- `PraticaControleEscolar`

## Tecnologias
- Java
- Eclipse IDE
- Git
